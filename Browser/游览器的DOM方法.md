# 游览器的DOM方法

DOM（Document Object Model）方法用于操作网页中的元素和结构。以下是一些常用的DOM方法：

1. **getElementById()**: 根据元素的ID获取该元素。
2. **getElementsByClassName()**: 根据元素的类名获取所有匹配元素的一个NodeList对象。
3. **getElementsByTagName()**: 根据标签名获取所有匹配元素的一个NodeList对象。
4. **querySelector()**: 根据CSS选择器获取匹配的第一个元素。
5. **querySelectorAll()**: 根据CSS选择器获取所有匹配元素的一个NodeList对象。
6. **createElement()**: 创建一个新的元素节点。
7. **appendChild()**: 将指定节点添加为当前节点的最后一个子节点。
8. **insertBefore()**: 在当前节点的子节点列表中，将指定节点插入到指定参考节点之前。
9. **removeChild()**: 将指定节点从当前节点的子节点列表中删除。
10. **setAttribute()**: 设置指定元素的属性值。
11. **getAttribute()**: 获取指定元素的属性值。
12. **classList**: 获取元素的类名列表，可用于添加、删除或切换类名。
13. **innerHTML / innerText / textContent**: 获取或设置元素内的内
